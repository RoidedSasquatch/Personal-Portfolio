My Courses

This page includes of a table consisting of the courses I have taken while in the Computer Programming and Analysis program at Algonquin. Courses in which exemptions were granted for previous degree courses are not included in this list. Use the options panel to search, sort, or filter through the table.

Image Course Level Code Description
Introduction to Computer Programming 1 CST8116

Introduction to Computer Programming is an introductory level course in Java Programming which teaches students the basics of Object-Oriented Programming and basic java constructs such as loops and arrays. Also provides an introduction to the planning process behind creating a program (UML, pseudocode).

Introduction to Database 1 CST8215

Introduction to Database teaches students the basics of database design, detailing the logic, conceptual, and physical design stages. It then teaches students how to create and alter databases through DDL and DML SQL code.

Computer Essentials 1 CST8118

Computer Essentials teaches students the basics of computers including hardware, machine code, storage specifics, backup and recovery, and basic linux operations. Familiarizes students with the use of VMWare and other VM tools.

Tech Math for Computer Science 1 MAT8001

Tech Math for Computer Science ensures that students have a required level of knowledge in mathematics concepts that would be used by programmers in the work force. Concepts include binary operations, trigonomoetry, algebra, and graphing functions.

Achieving Success In Changing Environments 1 CST8300

Achieving Success in Changing Environments prepares students for a career as a programmer by teaching soft skills, resume building, interview skills, and how to stay current in a rapidly changing industry. This course stresses the importance of communication in the workplace but also highlights the criticallity of hard skills.

Object Oriented Programming 2 CST8284

Object Oriented Programming teaches advanced Java techniques including Encapsulation, Inheritance, Abstraction, and Polymorphism. Concepts relating to threading and memory management are also discussed as well as the four access modifiers available in Java: public, private, protected, package.

Web Programming 2 CST8285

Web Programming covers topics pertaining to web development and how to maintain a website. Concepts covered in the course include HTML development, CSS development, writing Javascript, and PHP development. Students are taught about the various sections of a website, forms, and how data is sent to and from the server, as well as how the browser reads the code.

Operating Systems Fundamentals 2 CST8102

Operating System Fundamentals instructs students in the Linux terminal and Linux terminal operations. As part of this course, students create a new Ubuntu VM and disable the file explorer's GUI. Students are then taught numerous commands and syntax to allow filesystem operation from the terminal. Students also learn about how Linux operates on a system.

Database Systems 2 CST2355

Database Systems furthers students knowledge of SQL teaching intermediate concepts including triggers, roll and cube operations, and stored procedures. The course also educated students on various SQL distributions and management tools including MS Access, MS SQL Server, MS SQL Management Studio, MongoDB, and OracleDB.

Technical Communication for Engineering Tech. 2 ENL2019

Technical Communication for Engineering Technology enhances students ability to communicate with employers and more specifically, how to communicate technical concepts and conduct presentations for co-workers. The class also teaches students how to properly research and culminates in a research project worth the majority of the grade.

Coop Education and Job Readiness 2 GEP1001

Coop Education and Job Readiness prepares coop students for the job search process as well as how to conduct oneself while in the coop work terms. The course educates students on how to prepare an exceptional resume\ and cover letter, as well as how to be successful in the workplace.
