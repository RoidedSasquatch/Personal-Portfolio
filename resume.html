On this page, a copy of my resume can be found. Furthermore, a link to an academic programming project is included. This project was completed in Level 2 of the program and consists of a Java program that features inheritance as well as the use of arrays to model a car sales company.

CST8284 Inheritance Project

This project was completed for CST8284 Object Oriented Programming. The project contains a program which accepts inputs from the user and then determines if the user is eligible to drive a car and if they are pre-approved for a car purchase, by calculating their age and whether or not their salary is 80% greater than the purchase price of the car. It then prints this information to the console for the user to read. This is done by creating an instance of the class that contains the methods to store and save this data, and manipulating it. The program features the use of an array, several loops, and an inherited class.
